# Installation

## Stable release

To install MLOps Project, run this command in your terminal:

```sh
uv add mlops_project
```

Or if you prefer to use `pip`:

```sh
pip install mlops_project
```

## From source

The source files for MLOps Project can be downloaded from the [Github repo](https://github.com/NandiniKarnati21/mlops_project).

You can either clone the public repository:

```sh
git clone git://github.com/NandiniKarnati21/mlops_project
```

Or download the [tarball](https://github.com/NandiniKarnati21/mlops_project/tarball/master):

```sh
curl -OJL https://github.com/NandiniKarnati21/mlops_project/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd mlops_project
uv pip install .
```
